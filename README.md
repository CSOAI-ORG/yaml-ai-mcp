## Tools

### `validate_yaml`
Validate YAML syntax and report any errors.

**Parameters:**
- `content` (str)

### `convert_yaml_json`
Convert between YAML and JSON formats.

**Parameters:**
- `content` (str)
- `direction` (str)

### `lint_yaml`
Lint YAML content for style issues and best practices.

**Parameters:**
- `content` (str)

### `merge_yaml`
Merge two YAML documents together.

**Parameters:**
- `yaml_a` (str)
- `yaml_b` (str)
- `strategy` (str)
